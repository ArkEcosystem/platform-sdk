#meta: Record<string, TransactionDataMeta>
  • Defined in packages/platform-sdk/src/dto/transaction.ts:14

Various coins need post-processing to determine things like "isSent" or "isReceived" with data that comes from outside of the transaction or network data itself. This object can be used to store the data necessary for those actions.
